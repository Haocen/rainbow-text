Hello world!

Behold, this is the mighty rainbow-text!

To get started, add following content to your <head> tag:
<script src="https://haocen.github.io/rainbow-text/src/rainbow-text.js" type="module"></script>
And that's it! Now try to use <rainbow-text>Rainbow!<rainbow-text>

You can nest rainbow-text in any element that will change text style, for example:

  • You can use it in a list, like this.
  • You can use it any supported HTML symbols, just like this™.
  • Subscript, no problem.
  • Or, any other HTML elements just by nesting them.

Customization

You can assign CSS value to following variables:
--animation-duration, --animation-timing-function, --animation-iteration-count, --animation-direction, --animation-fill-mode, --animation-play-state
in order to fine tune the experience.
For example:

  • Make the rainbow flowing faster.
  • Make the rainbow flowing slower.
  • Make the rainbow static.

You can also set attribute flow-direction on the element:
For example:

  • Make the rainbow flowing from left to right, this is the default behavior.
  • Make the rainbow flowing from right to left is also possible.
